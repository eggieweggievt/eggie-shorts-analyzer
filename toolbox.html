The Toolbox 🧰

Little brain tools for the moments between the big tools — break a scary task into steps, fix the tone of a message, sanity-check how something reads, and get an honest time guess.

🤝 Honesty note: these run on hand-written rules and word lists, entirely in your browser — nothing you type leaves your machine, ever. They're a starting point, not magic. (Inspired by the wonderful goblin.tools — if you want the AI-powered versions, go hug a goblin.)

🪚 Task breaker

Type the scary task. Get small, physical steps. Pick how fine to chop it.

🎭 Tone shifter

Paste a message that doesn't sound right yet. Pick a direction. Edit the result — it's a draft, not gospel.

⚖️ Tone read

Worried how a message comes across? Get a heuristic read — and exactly which words triggered it, so you can decide for yourself.

⏳ Time guesser

A rule-of-thumb estimate with the real-brain buffer built in — because "5 minutes" is never 5 minutes.
