# Twitch Tags by Niche — Research Brief

*Compiled June 1, 2026. Companion to `twitch-tags-research.md`. Covers the 8 headline niches in the FAQ niche list / Growth Playbook. For the Twitch Tags playbook section.*

---

## How to read this

**Important caveat up front:** Twitch does **not** publish official tag popularity or "competition" numbers. The High / Medium / Low labels below are *estimates* synthesized from creator-growth resources (StreamRise, Overlay Forge, Hexeum, StreamHub, Algochat) and Twitch's own Creator Camp guidance — treat them as directional, not exact. The one source of truth for *your* channel is **Creator Dashboard → Analytics → Discovery → Tags**, which shows real impressions + click-through for the tags you actually use.

**Two rules that apply to every niche:**

1. **Identity stack goes on every stream:** `VTuber` always; add `ENVTuber` if you stream in English. These are your highest-value tags because the VTuber audience actively browses them.
2. **Lead with mid/low-competition tags, add 1–2 broad ones.** On a saturated tag you're buried; on a 50–500-channel niche tag you're actually visible. Use 5–7 genuinely relevant tags — relevance beats filling all 10.

The recommended shape per stream: **VTuber + ENVTuber + 1 category-true tag + 2–3 content/mood tags + (optional) 1 community tag.**

## 1. 🎮 Gaming

**Twitch category:** the specific game (e.g. *Stardew Valley*, *Valorant*) — Twitch auto-applies some category tags; your custom tags refine *genre, play-style, and experience*. Gaming is your biggest pool, so it's grouped below. Pick **one** genre tag, **one or two** play-style/experience tags, **one** mood tag, and a community tag if it fits — alongside `VTuber`/`ENVTuber`.

**A. Genre tags** (match the actual game — these pull genre-fans browsing across titles):

| Tag | Signals | Est. competition |
|---|---|---|
| `FPS` | First-person shooters | High |
| `RPG` / `JRPG` | Role-playing games | High / Medium |
| `MMORPG` / `MMO` | Massively multiplayer | Medium-High |
| `Soulslike` | Dark Souls / Elden Ring-likes | Medium |
| `Roguelike` / `Roguelite` | Run-based games | Medium |
| `Gacha` | Genshin, HSR, ZZZ, etc. (big VTuber overlap) | Medium-High |
| `CozyGaming` | Life-sim / farming / gentle | Medium |
| `Survival` / `ExtractionShooter` | Survival & extraction | Medium |
| `Metroidvania` | Exploration platformers | Low–Medium |
| `VisualNovel` / `DatingSim` | Story/VN games | Low–Medium |
| `Sandbox` / `Building` | Minecraft-likes, builders | Medium |
| `Strategy` / `Roguelike` / `Deckbuilder` | Tactical / card | Low–Medium |
| `Fighting` / `Platformer` / `Racing` | Genre signals | Low–Medium |

**B. Play-style / experience tags** (how you're playing — great for niche discovery):

| Tag | Signals | Est. competition |
|---|---|---|
| `FirstPlaythrough` | Genuine first reactions | Medium (strong for VTubers) |
| `NoSpoilers` | Spoiler-free, story-safe | Low–Medium |
| `BackseatGamingAllowed` / `NoBackseating` | Sets chat expectations | Low–Medium |
| `Casual` | Relaxed, low-pressure | Medium |
| `Competitive` / `Ranked` / `PvP` | Tryhard / ranked play | Medium-High |
| `PvE` / `CoOp` | Co-op / vs-environment | Medium |
| `Completionist` / `100Percent` | Full clears | Low–Medium |
| `Hardcore` / `ChallengeRun` | High-difficulty runs | Low–Medium |
| `Modded` / `Randomizer` | Modded or randomized | Low–Medium |
| `Nuzlocke` | Pokémon challenge ruleset | Low (niche) |
| `Speedrun` | **Only** if genuinely speedrunning | Low (niche, protective community) |
| `BeginnerFriendly` | Welcoming to new viewers | Low–Medium |

**C. Community & interaction tags:**

| Tag | Signals | Est. competition |
|---|---|---|
| `PlayingWithViewers` / `ViewerGames` | Viewers can join | Medium |
| `Charity` | Charity fundraiser stream | Low (event) |
| `Subathon` | Subathon event | Low–Medium |
| `DropsEnabled` | Auto-added during Drops campaigns | Very High (don't add manually) |

**D. Mood / vibe tags** (pair with any of the above):

| Tag | Signals | Est. competition |
|---|---|---|
| `Chill` | Relaxed energy | Medium-High |
| `Cozy` | Soft, comfy | Medium |
| `Chatty` | Talk-while-playing | Medium |
| `HighEnergy` | Hype / loud | Medium |
| `Funny` / `GoodVibes` | Comedy / positive | Medium-High |

**Avoid:** `Gaming`, `Live`, `Twitch`, `ProGamer` — too broad or no search intent. Don't use `Speedrun`/`Competitive`/`Nuzlocke` unless it's literally true (mismatched viewers bounce, which hurts you). `DropsEnabled` is applied automatically — no need to type it.

**Example sets (VTuber):**
- Cozy farming sim: `VTuber` `ENVTuber` `CozyGaming` `Cozy` `FirstPlaythrough` `NoSpoilers`
- Gacha daily: `VTuber` `ENVTuber` `Gacha` `RPG` `Chatty` `PlayingWithViewers`
- Soulslike blind run: `VTuber` `ENVTuber` `Soulslike` `FirstPlaythrough` `NoSpoilers` `HighEnergy`
- Ranked shooter: `VTuber` `ENVTuber` `FPS` `Competitive` `Ranked` `Chatty`
- Co-op chaos w/ friends: `VTuber` `ENVTuber` `CoOp` `PlayingWithViewers` `Funny` `Casual`

**Twitch category:** *Just Chatting* — the single most saturated category, so tags + title + personality do the heavy lifting. Pick a format/topic tag, an engagement tag, a community tag, and a mood tag — plus `VTuber`/`ENVTuber`.

**A. Format / topic tags:**

|---|---|---|
| `AMA` | Ask Me Anything / Q&A | Medium |
| `Storytime` | Narrative chatting | Low–Medium |
| `Reaction` / `WatchParty` | Reacting to content | Medium |

## 7. 👹 Horror & spooky

**Twitch category:** the specific horror game (*Phasmophobia*, *Lethal Company*, *Dead by Daylight*, *Resident Evil*, *Outlast*, etc.); custom tags add the experience. Pick a sub-genre tag, a play-style tag, a mood/seasonal tag, plus `VTuber`/`ENVTuber`. Several of these have live directory pages (linked below).

**A. Sub-genre tags:**

| Tag | Signals | Est. competition |
|---|---|---|
| `Horror` | General horror | High |
| `Scary` | Fear-focused | High |
| `Spooky` | Lighter/atmospheric | Medium-High |
| `HorrorGaming` / `ScaryGames` | Horror games specifically | Medium-High / Medium |
| `SurvivalHorror` | Resource/survival mechanics | Medium |
| `IndieHorror` | Indie titles (underused) | Medium (good niche) |
| `PsychologicalHorror` | Atmospheric/mind-bending | Low–Medium |
| `Jumpscares` | Reaction hook | Medium |

**B. Play-style / experience tags:**

| Tag | Signals | Est. competition |
|---|---|---|
| `FirstPlaythrough` / `NoSpoilers` | Fresh, story-safe | Medium / Low–Medium |
| `Reactions` / `ScaredyCat` | Showcases avatar reactions | Medium / Low |
| `CoOp` | Group horror (Phasmo, Lethal Co.) | Medium |
| `Blind` / `FirstTime` | First exposure (see note) | Low–Medium |

**C. Mood / seasonal tags:** `Spooky`, `Halloween` (Oct spike), `HighEnergy`, `Chatty`, `Cozy` (cozy-horror crossover only).

**Note:** the old `BlindPlaythrough` tag was **retired by Twitch in 2020** for accessibility reasons — use `FirstPlaythrough` + `NoSpoilers` (or plain `FirstTime`) instead. October = `Spooky`/`Halloween`/`Horror` seasonal spike.

**Avoid:** stacking three very-high-competition tags (`Horror` + `Scary` + `VTuber`); non-standard tags like `Terrified`.

**Example sets (VTuber):**
- Co-op scares: `VTuber` `ENVTuber` `Horror` `CoOp` `Jumpscares` `Reactions`
- First-time story horror: `VTuber` `ENVTuber` `SurvivalHorror` `FirstPlaythrough` `NoSpoilers` `Reactions`
- Indie horror night: `VTuber` `ENVTuber` `IndieHorror` `PsychologicalHorror` `Spooky` `Chatty`

*Sources: Twitch tag directories ([Horror](https://www.twitch.tv/directory/all/tags/Horror), [ScaryGames](https://www.twitch.tv/directory/all/tags/ScaryGames), [FirstPlaythrough](https://www.twitch.tv/directory/all/tags/FirstPlaythrough), [NoSpoilers](https://www.twitch.tv/directory/all/tags/NoSpoilers)), [StreamRise](https://stream-rise.com/blog/guide-to-tags).*
